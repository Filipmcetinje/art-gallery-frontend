# Art Gallery Explorer - Frontend

A React-based web application that allows users to browse artworks from The Metropolitan Museum of Art API, view artwork details, and (in later stages) save favorite artworks to a personal collection.

## Live demo

https://filipmcetinje.github.io/art-gallery-frontend/

## Features

- Search artworks from the Met Museum Collection API
- View artwork preview in a modal
- Browse results and load more items
- Saved artworks page (UI)

## Tech stack

- React
- Vite
- React Router
- Fetch API
- CSS (BEM)
